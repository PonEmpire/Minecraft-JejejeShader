# Minecraft-JejejeShader

統合版用のシェーダー、いわゆる影modを作っていきます。
現段階では正式リリースはありません。プレリリース版があります。
## 美しくなる点
・雲
・太陽
・月
・溶岩
・雨
・水
・ガラス
## 導入方法
mcpackファイルをダウンロードし、ダブルクリックで
マインクラフトが開きます。そして、ワールドの編集の
ところで影modを使用できます。Ninetendo switchにも、
Realmsを使用することで導入することができます。

## サポートされている機種
